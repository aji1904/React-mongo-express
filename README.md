# React-mongo-express

install on folder Server
- install jsonwebtoken
- install express
- install bcrypt
- install mongoose
- install cors
- install router

install on folder Client
- install react-center
- install material-ui
- install axios
- npm i simple-webstorage --save

run server on folder Server
- cd Server
- npm run dev

run client on folder Client
- cd Client
- npm start
